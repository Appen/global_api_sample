Welcome to Appen Global API

This document describes the RESTful Appen Global API v2. In addition we have supplied some very simple examples showing how to access the API using Ruby. Accessing the API is not limited to Ruby as it is a RESTful API passing data using Json.

Appen Global API is currently at version 'v2'.

Introduction

This document is intended for Appen clients who need to programmatically interact with the Appen Global (Global). With Global's API, you can submit content to be analyzed and then retrieve the results after work on the data has been completed.

AGP API's are based on the RESTful concept and use JSON containers both for message submission and for returning results.
